# DSA Visualizer

This project is a simple web‑based tool to visualize **Linear and Non‑Linear Data Structures**.  
It is built using **HTML, CSS, and JavaScript** to help learners understand how data is organized and accessed.

---

## 📂 Data Structures Covered
- **Linear Data Structures**  
  - Array  
  - Stack  
  - Queue  
  - Linked List  

- **Non‑Linear Data Structures**  
  - Tree  
  - Graph  

---

## 🚀 Features
- Interactive visualization of linear and non‑linear structures.  
- Step‑by‑step animations to show insertion, deletion, and traversal.  
- Minimal UI for easy learning.  

---

## 🛠️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Kommireddy19/DSA-visualizer.git
   ```
2. Open `index.html` in your browser.  
3. Select a data structure to visualize.
